BeginPython笔记:6.Dictionary

Posted on 2018-02-27 | In Python , BeginPython笔记

本文是《Python基础教程(第2版 修订版)》第 4 章 字典 的笔记，简要介绍了字典的创建、基本字典操作、字典的格式化字符串和字典方法。

1. dict

  • dict函数可以通过映射或者键/值对的序列建立字典，也可以通过关键字参数来创建字典。不带任何参数时，则返回一个新的空字典。

2. Basic Dictionary Operations

  • len(d)返回d中项（键-值对）的数量。
  • d[k]返回关联到键为k上的值。
  • d[k]=v将值v关联到键k上。
  • del d[k]删除键为k的项。
  • k in d检查d中是否有含有键为k的项。
  • 键类型：字典的键可以是任意的不可变类型。

4. Dictionary Methods

4.1 clear

  • clear方法清除字典中所有的项。无返回值。
  • clear方法能够清空本身、其关联的字典和原始字典。

4.2 copy

  • copy方法返回一个相同键-值对的新字典。该方法实现的是浅复制。当在副本中替换值时，原始字典不会受到影响。但如果修改了某个值时，原始字典也会发生改变。
  • 可以使用copy模块中的deepcopy函数进行深度复制以避免该问题。

4.3 fromkeys

  • fromkeys方法使用给定的键建立新的字典，对应的默认值为None，也可以自己提供默认值。

4.4 get

  • 一般来说，当访问字典中不存在的项时会报错。而使用get方法时，则不会，会返回None，也可以自定义默认值。

4.5 has_key

  • has_key方法可以检查字典中是否含有特定的键，同k in d。Python3中不包括这个函数。

4.6 items、iteritems

  • items方法将字典所有的项以列表方式返回，列表中的每一项都表示（键，值）对的形式。但是项在返回时并不遵循特定的次序。
  • iteritems方法则返回一个迭代器对象，而非列表。

4.7 keys、iterkeys

  • keys方法将字典中的键以列表的形式返回。iterkeys则返回针对键的迭代器。

4.8 pop

  • pop方法用来获得对应于给定键的值，然后将这个键-值从字典中移除。

4.9 popitem

  • popitem方法会随机删除字典中的项，并返回项。

4.10 setdefault

  • setdefault方法在键不存在的时候，会返回默认值，同时相应地更新字典。若存在，则返回值，且不改变字典。默认值默认为None，可修改。

4.11 update

  • update方法利用一个字典项更新另一个字典。提供的字典中的项会被添加到旧的字典中，而相同的键会进行覆盖。
  • update方法可以使用与调用dict函数（或者类型构造函数）同样的方式进行调用，即uodate方法可以和映射、拥有（键-值）对的队列（或者其他可迭代的对象）以及关键字参数一起调用。

4.12 values、itervalues

  • values方法以列表形式返回字典中的值，且可以包含重复的元素。itervalues方法则返回值的迭代器。
