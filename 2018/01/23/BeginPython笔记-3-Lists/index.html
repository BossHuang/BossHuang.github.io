BeginPython笔记:3.Lists

Posted on 2018-01-23 | In Python , BeginPython笔记

本文是《Python基础教程(第2版 修订版)》第 2 章 列表 部分的笔记，简要介绍了列表的基本操作和常用方法。

1. The List

  • 因不能直接修改字符串，所有有时需要根据字符串创建列表
  • List适用于所有类型的序列，而不只是字符串。

2. Basic List Operations

列表可以使用所有适用于序列的标准操作。虽然列表是可修改的，但是并不是所有的列表方法能真正地改变列表。

2.1 Item assignments

2.2 Delete elements

2.3 Assigning to slices

  • 在分片赋值时，可以使用与原序列不等长的序列进行分片替换。
  • 分片赋值语句可以在不替换任何元素下插入新的元素。
  • 可以通过分片赋值来删除元素。

3. List Methods

方法是一个与某些对象有紧密联系的函数，对象可能是列表、数字，也可能是字符串或者其他类型的对象。方法一般这么调用：对象.方法(参数)

3.1 append

  • append方法直接修改原来的列表，而不是返回一个修改过的新列表。

3.2 count

  • count方法统计某个元素在列表中出现的次数。

3.3 extend

  • extend方法可以在列表的末尾一次性追加另一个序列中的多个值。
  • extend方法修改被扩展的序列，而连接操作会返回一个新的列表。
  • 在这个，如果进行 a=a+b 的连接操作，效率会比extend方法低。

3.4 index

  • index方法可以从列表中找出某个值第一个匹配项的索引位置。
  • 如果没有被找到时，会引发一个异常。

3.5 insert

  • insert方法用于将对象插入到列表中。

3.6 pop

  • pop方法会移除列表中的一个元素，默认是最后一个，且返回该元素的值。
  • pop方法是唯一一个既能修改列表，又能返回元素值（除了None）的列表方法

3.7 remove

  • remove方法用于移除列表中某个值的第一个匹配项。
  • remove方法改变原列表，且无返回值。
  • 移除一个不存在的值，会引发一个异常。

3.8 reverse

  • reverse方法将列表中的元素反向存放。该方法改变列表，但无返回值。
  • 如果需要对一个序列进行反向迭代，则可以使用reversed函数。该函数并不返回一个列表，而是返回一个迭代器对象。

3.9 sort

  • sort方法会在原位置上对列表进行排序，而返回空值。
  • 如果需要一个排序好的列表副本，又同时保留原列表不变，则先进行深度拷贝或者使用sorted函数。
  • sorted函数可以用于任何可迭代对象，总是返回列表。

3.10 advanced sort

  • 如果希望元素按照某种特定的方式排序，可以通过compare(x, y)的形式自定义比较函数。conpare(x, y)函数在 x > y 时返回正数，x < y时返回负数，x==y时返回0。定义好该函数后可以提供给sort方法作为参数。内建cmp函数提供了比较函数的默认实现方式。
  • sort方法的key参数，需要提供一个排序中使用的函数，为每个元素创建一个键，再通过键进行排序。
  • sort方法的reverse参数，True为降序，False为升序，默认为False
  • sort方法的cmp、key、reverse参数都可以用于sorted函数。
