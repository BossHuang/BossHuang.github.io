BeginPython笔记:5.String

Posted on 2018-01-26 | In Python , BeginPython笔记

本文是《Python基础教程(第2版 修订版)》第 3 章 字符串 的笔记，简要介绍了字符串基本操作、字符串格式化和字符串方法。

1. Basic String Operations

  • 所有标准的序列操作，如索引、分片、乘法、判断成员资格、求长度、取最大值和最小值，均适用。
  • 字符串是不可变的，所以所有像分片赋值这样的操作是不合法的。

2. String Formatting

  • 字符串格式化需要使用字符串格式化操作符，即%来实现。要在格式化字符串里输出%，则需要使用%%。
  • 只有元组和字典可以格式化一个以上的值，而列表或者其他的序列都会被解释为一个值。
  • string模块提供模块字符串来格式化值。
    • substitute模块方法会用传递进来的关键字参数foo替换字符串中的$foo
    • 如果替换字段是单词的一部分，那么参数名就必须用花括号括起来。
    • 可以使用$$插入美元符号。
    • 除关键字参数外，还可以使用字典变量提供值/名称对。
    • 方法safe_substitute不会因缺少值或者不正确使用$字符而报错。
  • 基本的转换说明符：
    1. %字符：标记转换说明符的开始。
    2. 转换标志：可选。
      • -表示左对齐。
      • +表示在转换之前要加上正负号
      • “ ”(空白字符)表示正数之前保留空格。
      • 0表示转换值若位数不够则用0填充。
    3. 最小字段宽度：可选。转换后的字符串至少应该具有该值指定的宽度。如果是*，则宽度从值元组中读出。
    4. 点.后跟精度值：可选。如果转换的是实数，精度值就表示出现在小数点后的位数。如果转换的是字符串，则表示最大字段宽度。如果是 *，则精度从元组中读出。
    5. 转换类型：
      • d, i: 带符号的十进制整数
      • o: 不带符号的八进制
      • u: 不带符号的十进制
      • x, X: 不带符号的十六进制（小写，大写）
      • e, E: 科学计数法表示的浮点数（小写，大写）
      • f, F: 十进制浮点数
      • g, G: 如果指数大于-4或者小于精度值则和e, E相同，其他情况与f, F相同。
      • C: 单字符（接受整数或者单字符字符串）
      • r: 字符串（使用repr转换任意对象）
      • s: 字符串（使用str转换任意对象）

3. String Method

  • 虽然字符串方法都来源于string模块，但其中仍有一些不能作为字符串方法使用的常量和方法。例如，一些有用的字符串常量：
    • string.digits: 包含数字0~9的字符串
    • string.letters: 包含所有字母（大写或小写）的字符串。
    • string.lowercase: 包含所有小写字母的字符串。
    • string.uppercase: 包含所有大写字母的字符串
    • string.printable: 包含所有可打印字符的字符串
    • string.punctuation: 包含所有标点的字符串。
    • 字母字符串常量与地区有关，即Python所匹配的语言。如果使用的是ASCII，那么可以在变量中使用ascii_ 前缀。Python3中，string.letters和其他内容都会被移除。可以使用string.ascii_letters代替。

3.1 find

  • find方法可以在一个较长的字符串中查找子串，返回子串所在位置的最左端索引。如果没有找到，则返回-1。可以指定起点和终点。
  • in操作符只能查找字符串中的单个字符。

3.2 join

  • 用来连接序列中的元素。被连接的序列元素必须是字符串。是split方法的逆方法。

3.3 lower

  • lower方法返回字符串的小写字母版。
  • title方法会将字符串转换为标题，即所有单词的首字母大写，其他的小写。string模块的capword函数也提供同样的功能。但是对首字母大写的规则不同。

3.4 replace

  • replace方法返回某字符串的所有匹配项均被替换后得到的字符串。

3.5 split

  • split方法用来将字符串分割成序列。如果不提供任何分隔符，则会把所有空格作为分隔符（空格、制表、换行等）

3.6 strip

  • strip方法返回去除两侧（不包括内部）空格或指定字符后的字符串。

3.7 translate

  • translate方法可以替换字符串中的某些部分，与replace方法不同的是，该方法只处理单个字符。优势在于可以同时进行多个替换，有时效率会比replace方法高。
  • translate方法的第二个参数是可选的，用来指定需要删除的字符。
