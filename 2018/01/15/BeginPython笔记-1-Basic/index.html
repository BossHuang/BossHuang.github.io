BeginPython笔记:1.Basic

Posted on 2018-01-15 | In Python , BeginPython笔记

本文是《Python基础教程(第2版 修订版)》第 1 章 基础知识的笔记，简要介绍了Python中的基础知识，如变量、数字、表达式、获取用户输入、模块导入和命令行执行等。

1. Begin

  • Python 是一种解释型、面向对象、带有动态语义的高级程序设计语言
  • Jython利用Java实现，运行在虚拟机中。IronPython则使用C#实现，运行于公共语言运行时(Common Language Runtime, CLR)的.NET和MONO实现。

2. Numbers and Expressions

  • 单斜线根据两边的数据类型执行不同的操作。双斜线只做整除。
  • 取余运算符(%)
  • 幂运算符
  • 普通整数大于2147483647(2**31-1)或小于-2147483648，Python按照长整型数据类型处理。
  • 16进制和8进制

3. Variables and Statements

  • 变量基本上就是代表（或者引用）某值的名字。
  • 变量名可以包括字母、数字和下划线(_)。但是不能以数字开头。
  • 表达式就是某件事情，语句是做某件事情，即告诉计算机做什么。

4. Getting input from the User

  • input假设用户输入的是合法的Python表达式
  • raw_input则把所有的输入当作原始数据(raw data)
  • 一般来说尽可能使用raw_input函数，除非对input有特殊需求。

5. Modules

  • 通过import 模块导入模块，则模块.函数方式进行调用。另外也可以通过变量来引用函数。
  • 通过from 模块 import 函数，则可直接使用函数。但是可能会存在命名冲突。

6. Executing Programs

  • 命令行执行脚本：python hello.py
  • 在UNIX中另一种实现方法：在首行前面加上#!，而后加上用于解释脚本的程序的绝对路径。例如：#! /usr/bin/env python。再通过chmod a+x hello.py使脚本文件具有可执行属性。

7. Strings

  • 单引号和双引号都存在的一个用处
  • str、repr和反引号` 可以把Python值转换为字符串：
    • str本质是一种类型，会把值转换成合理形式的字符串，以便用户理解。
    • repr函数会创建一个字符串，以合法的Python表达式的形式来表示值。
    • 反引号在Python 3中已被弃用。
  • 长字符串：需要一个非常长的字符串，需要跨多行，可以采用：三个引号，如'''hello world'''，或者三个双引号，如"""hello world"""。在其中可以同时使用双引号和单引号，且不需要反斜杠进行转义。
  • 原始字符串：以r开头，可以放入任何字符，但最后一个字符不能是反斜杠。
  • Unicode字符串：使用u前缀。Python中普通的字符串在内部是以8位的ASCII码存储，Unicode字符串是以16位Unicode字符存储。在Python 3中，所有字符串都是Unicode字符串。
