BeginPython笔记:2.Sequence

Posted on 2018-01-17 | In Python , BeginPython笔记

本文是《Python基础教程(第2版 修订版)》第 2 章 序列 部分的笔记，简要介绍了序列的基本概念和常用的基本操作。

1. Overview

  • 序列是Python中最基本的数据结构。序列中的每一个元素都被分配一个序号，即索引，元素的位置，从0开始。
  • Python中有一种名为容器（container）的数据结构。容器基本上是一种包含其他对象的任意对象。序列（例如列表和元组）和映射（例如字典）是两类主要的容器。集合是既不是序列也不是映射的容器类型。
  • Python包含 6 种内建的序列：列表、元组、字符串、Unicode字符串、buffer对象和xrange对象。
  • 列表和元组的主要区别在于，列表可修改，元组不能。因此序列不能修改时，使用元组更合适。

2. Common Sequence Operations

  • 所有序列类型都可以进行某些操作：索引（indexing）、分片（slicing）、加（adding）、乘（multiplying）和成员资格检查。此外，还有计算序列长度、求最大值最小值等内建函数。
  • 迭代（iteration）：依次对序列中的每个元素重复执行某些操作。
  • 序列可以包含其他序列

2.1 Indexing

  • 使用负数索引时，从最后一个元素开始计数。
  • 函数调用返回的是序列，则可以直接对返回结果进行索引操作。

2.2 Slicing

  • 分片操作需要提供两个索引作为边界，第一个索引的元素会包含在分片内，而第二个则不包含在内。
  • 分片中最左边的索引比右边的索引晚出现在序列中，则返回一个空的序列。
  • 可以设置步长，通常是隐式设置。步长不能为0，可以为负数，此时分片从右向左提取元素。

2.3 Adding

  • 相同类型的序列才能进行加操作。

2.4 Multiplication

  • 数字 x 乘以一个序列会生成新的序列，且在新序列中，原来的序列会被重复 x 次。
  • 可以用 None 创建占用一定空间但是不包括任何有用内容的列表。

2.5 Membersship

  • 使用 in 运算符检查一个元素是否在序列中

2.6 Length, Minimum and Maximum

  • len 函数返回序列中所包含元素的数量
  • min 函数和 max 函数分别返回序列中的最小和最大的元素，其参数并不是序列，而是以多个数字直接作为参数
